# 🌿 VedTrim — Ayurvedic Weight Loss & Wellness Platform

VedTrim is a modern, cloud-powered Ayurvedic wellness platform that combines ancient wisdom with cutting-edge technology to deliver personalized weight management solutions.

## ✨ Features

- **Personalized Dosha Quiz** — Discover your Ayurvedic body type (Vata, Pitta, Kapha)
- **Expert Vaidya Consultations** — Connect with certified Ayurvedic practitioners
- **Ayurvedic Medicine Catalog** — Browse traditional formulations for weight wellness
- **Client Dashboard** — Track your weight loss journey with daily progress charts
- **Vaidya Portal** — Dedicated dashboard for practitioners to manage clients
- **AI Chatbot** — Get instant Ayurvedic wellness guidance
- **Secure Payments** — Integrated payment system for subscriptions

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Design**: Glassmorphism, responsive mobile-first design
- **Hosting**: GitHub Pages
